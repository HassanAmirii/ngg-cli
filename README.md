A classic number guessing game built as a command-line interface application using Node.js. The game challenges you to guess a randomly generated number between 1 and 100, offering multiple difficulty levels to test your intuition.

## Features

-   **Multiple Difficulty Levels**: Choose from Easy (10 attempts), Medium (5 attempts), or Hard (3 attempts) to match your skill.
-   **Interactive Gameplay**: Provides real-time feedback, telling you if your guess is too high or too low.
-   **Input Validation**: Gracefully handles non-numeric inputs, prompting the user to enter a valid number.
-   **Attempt Tracking**: Keeps a count of your guesses, celebrating your efficiency upon winning.
-   **Replayability**: Offers the option to start a new game immediately after one ends.

## Technologies Used

| Technology | Description |
| :--- | :--- |
| [Node.js](https://nodejs.org/) | An asynchronous event-driven JavaScript runtime, used to build the game's logic and command-line interface. |
| [Readline](https://nodejs.org/api/readline.html) | A native Node.js module used to handle readable streams one line at a time, perfect for capturing user input in the terminal. |

## Usage

To run the game, you first need to have Node.js installed on your machine.

1.  Save the code as `index.js`.
2.  Open your terminal or command prompt.
3.  Navigate to the directory where you saved the file.
4.  Run the game with the following command:

    ```bash
    node index.js
    ```

5.  The game will start, and you will be prompted to select a difficulty level.
